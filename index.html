Sobre o Projeto

Este projeto oferece um simulador de circuito RLC robusto, permitindo a análise com fontes de corrente contínua (DC) e alternada (AC). Ao final de cada simulação, um gráfico detalhado da corrente em função do tempo é gerado, proporcionando uma visualização clara e precisa do comportamento do circuito.

Desenvolvido em Java com a biblioteca JavaFX para uma interface gráfica rica e intuitiva, o simulador utiliza o método de Runge-Kutta de 4ª ordem para a resolução da equação diferencial do circuito, garantindo resultados precisos.

Instalação e Execução

Para começar a usar o simulador, siga estes simples passos:

  1. Instale o Java Development Kit (JDK) 21:
    É crucial ter o JDK 21 ou superior. Baixe-o diretamente do site da Oracle e siga o assistente de instalação.
  2. Baixe o Kit do Simulador:
    Clique nos botões abaixo para baixar os arquivos necessários.
    3. SimuladorRLC.bat (Para Windows)
    SimuladorRLC.sh (Para macOS/Linux)
    FisicaIII-1.0-SNAPSHOT.jar (Arquivo do programa)
  3. Execute o Simulador:
    • Windows: Salve todos os arquivos na mesma pasta e execute SimuladorRLC.bat.
    • macOS e Linux: Salve os arquivos na mesma pasta, abra o terminal e execute: sh SimuladorRLC.sh
